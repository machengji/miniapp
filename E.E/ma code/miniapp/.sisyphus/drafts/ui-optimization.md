# Draft: UI 优化规划

项目：Aletheia（荣格心理学小程序）
日期：2026-01-31

## 用户需求
- 优化整体UI

## 当前UI状态分析

### 页面结构
1. **index** - 首页/梦境日记
   - 统计卡片（已做渐变设计）
   - 对话历史卡片列表
   - 梦境记录卡片列表
   - 悬浮按钮（FAB渐变+脉冲动画）

2. **chat** - 聊天页面
   - 思维链可视化（独特设计）
   - 消息气泡
   - 输入区域（毛玻璃效果）
   - 元数据弹窗

3. **report** - 心灵报告页
   - 雷达图（Canvas绘制）
   - Persona/Shadow极性条
   - 情绪分布条形图
   - 意象云标签

4. **detail** - 梦境详情页
   - 卡片式布局
   - 思考过程展开
   - 原型分布列表

### 组件
1. **navigation-bar** - 自定义导航栏
2. **daily-card** - 每日共时性卡片（带3D翻转效果）

## 现有设计优点
- 暗色主题统一（#121212背景）
- 紫色主色调（#8a6dff, #6e00ff）符合心理学神秘感
- 毛玻璃效果（backdrop-filter）
- 渐变按钮和卡片
- 3D翻转卡片（daily-card）
- Canvas雷达图
- 脉冲动画、浮动动画

## 潜在优化方向

### 1. 动画增强
- 页面转场动画
- 卡片列表进入动画（stagger效果）
- 滚动淡入效果
- 水波纹点击效果
- 加载骨架屏

### 2. 交互优化
- 长按反馈增强
- 滑动操作（左滑删除/操作）
- 手势导航
- 触觉反馈（vibrateShort）

### 3. 视觉精细度
- 字体层级优化
- 间距更规范
- 阴影层次更丰富
- 图标统一
- 表情符号系统化

### 4. 性能优化
- 懒加载图片/组件
- 减少重绘
- 动画使用transform/opacity
